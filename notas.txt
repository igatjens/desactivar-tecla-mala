Detectar tecla

Desactivar tecla

Script para desactivar

! Disable Key
keycode XX = NoSymbol

Ejecutar script

xmodmap key_script




Script para reasignar

! Remap Key
keycode XX = NewFunction


Ejecutar script

xmodmap key_script
